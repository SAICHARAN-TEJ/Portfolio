# Portfolio

A personal portfolio website showcasing my work and projects. The site is hosted on Neocities and features two versions with different designs.

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom CSS with CSS variables for theming
- **Three.js** - 3D graphics and visual effects
- **GSAP + ScrollTrigger** - Smooth animations and scroll-based interactions
- **Google Fonts** - Syne & DM Mono typography
